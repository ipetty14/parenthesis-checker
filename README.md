# Parenthesis Checker

A basic parenthesis checker for string syntax. Specifically designed for need in LISP.
## Getting Started

### Dependencies

* Node
* NPM

### Installing

* Clone repository
* Run `npm install`

### Executing program

* Once installed run the following command:
```
npm start
```

### Testing

* To run the test suite, run the following command:
```
npm run test
```
